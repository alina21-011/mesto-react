# Проект: Mesto-React

### Обзор
Проект по созданию  одностраничного сайта на React.

**Интро**

Проект представляет собой сайт, где можно посмотреть карточки фотографий , добавить новые или удалить. 
Реализованы несколько попапов для редатирования данных.

HTML | CSS | Java Script | Webpack | API | React
